# 低于市场价！重庆法拍房的便宜不是那么好占的

> 来源：[[猫评楼市]] | 日期：2025-07-04 | 公众号文章

## 全国市场（2025年1-5月）

| 指标 | 数据 |
|------|------|
| 挂拍量 | **32.4万套**（同比微降0.78%） |
| 成交量 | **6.487万套** |
| 成交金额 | **1016.9亿元** |
| 特征 | 成交集中于一线城市核心地段豪宅 |

## 重庆市场数据

### 2024全年
- 挂拍：**38,535套**
- 月均成交：870套
- 成交额：**92亿元**

### 2025年5月
| 指标 | 数据 |
|------|------|
| 上拍 | 799套（环比+20%） |
| 成交 | 339套，成交率**42%** |
| 面积 | 3.76万㎡ |
| 建面均价 | **7,938元/㎡**（环比+6%） |
| 金额 | 2.98亿元 |

### 区域分布
- **渝北区**：挂拍165套，成交79套（双第一）
- **南岸区**：挂拍119套，成交60套，成交率50%
- **渝中区**：成交率最高，达**56%**

## 法拍房优势

1. **价格优势**：起拍价≈市价70%-80%，二拍可降至6折。全国法拍住宅平均折价率**86.7%**（8.7折）
2. **突破限购**：多数城市不限购
3. **产权清晰**：法院强制确权，凭《执行裁定书》快速过户
4. **税费较低**：部分仅需缴契税

## 七大风险

| 类型 | 内容 |
|------|------|
| 产权风险 | 产权不清晰、权属纠纷 |
| 费用风险 | 手续费/税费高，物业费水电费欠缴 |
| 交房风险 | 无法按时交房，被执行人不配合 |
| 贷款风险 | 部分需一次性付清，银行审批严格 |
| 法律风险 | 拍卖程序不规范 |
| 税收风险 | 税费承担主体不明确 |
| 户口风险 | 原房主不迁户口，无法落户/上学 |

## 结论

法拍房价格优势明显，但需全面调查产权、费用、占用情况，**"便宜不是那么好占的"**。
