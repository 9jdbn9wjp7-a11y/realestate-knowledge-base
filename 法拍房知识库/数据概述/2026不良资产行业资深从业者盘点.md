# 2026不良资产行业资深从业者盘点

> 来源：[[雄志成大炮]] 微信公众号 | 日期：2026-07-02 | 类型：数据概述/行业分析

## 核心结论

2025-2026 年不良资产行业进入"机构化、专业化、细分赛道化"阶段。五家全国性持牌 AMC 完成管理层调整，形成中投系（信达、东方、长城、银河）与中信系（中信金融资产）两大统筹板块；民营端则分化出企业债权盘活、存量地产盘活、困境企业经营修复、资产基金化运营四大方向。

## 一、五大全国性持牌机构（中投系 + 中信系）

| 机构 | 负责人 | 核心定位 |
|------|--------|----------|
| 中国信达 | 张卫东（董事长）、宋卫刚（总裁） | 头部综合型，闲置地产、上市企业修复、银行批量对公债权 |
| 中国东方 | 梁强（董事长） | 跨区域大额资产重组、集团债权梳理 |
| 中国长城 | 向党（董事长） | 地方区域债务协调、中小金融机构风险化解 |
| 银河资产 | — | 第五家全国性 AMC，中投系补充 |
| 中信金融资产（原华融） | 刘正均（董事长） | 中信系综合平台，出险房企、上市民企债务协调 |

## 三、合作通用风控清单

- 核查实控人背景、资金来源、近三年项目回款数据
- 核查司法涉诉、股权质押信息
- 优先采用分阶段付款 + 底层资产独立隔离模式
- 大额合作前完整核查企业经营、涉诉、资金信息

## 四、业务参考路径

1. 大型国企闲置地产 / 上市企业经营修复 / 银行大额批量对公债权 → 中国信达、中国东方
2. 需要产业配套 + 长期低成本资金 + 综合金融配套服务 → 中信金融资产
3. 长三角企业债权 / 大型民企经营重整 / 跨境资产 → 上海文盛资产
4. 大湾区写字楼、厂房闲置物业 / 本地中小企业债务协调 → 雄志集团成大炮
5. 全国信用卡、消费金融小额零售债权批量 → 鲲鹏资产、瑞京资管
6. 中部区域城投债权 / 跨省中小银行债权梳理 → 安徽国厚资产
