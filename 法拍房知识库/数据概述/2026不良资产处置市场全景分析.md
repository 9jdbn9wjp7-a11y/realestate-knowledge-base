# 2026不良资产处置市场全景分析

> 来源：[[华君策]]｜2026-07-04｜[[【金融行研系列-7】不良资产处置市场2026全景分析报告]]
> 类型：行业研究 / 市场数据

## 核心论断

2026年中国不良资产处置市场正处于从**"规模扩张"向"价值重构"**转型的关键周期。

## 关键数据

| 指标 | 数据 |
|------|------|
| 2026年行业整体市场规模 | **5.18万亿元** |
| 2030年预测市场规模 | **突破8万亿元** |
| 涉房类不良资产占比 | **30%以上** |
| 非银金融机构风险资产年增速 | **25%** |
| 四大国有AMC合计市场份额（2025） | **57.1%** |
| 中信金融资产2026H1处置招商债权 | **741.8亿元**（占31.76%） |
| 消费金融公司不良资产转让同比增幅（2026Q1） | **96.6%** |
| 阿里/京东2026H1撮合交易占比 | **超三成** |

## 资产供给结构

- 商业银行不良贷款：**50%**（核心供给，占比下滑）
- 非银金融机构不良资产：**30%**（年增25%）
- 非金特殊资产：**20%**（房地产、城投相关占六成）

## 处置模式演变

1. **重组重整模式** → 主流核心路径
2. **资产证券化模式** → 创新退出路径（Pre-REITs、不良ABS）
3. **互联网平台处置模式** → 标准化流量路径
4. **债权转让模式** → 快速变现，利润收窄

## 区域分布

- 长三角、珠三角、京津冀2025年合计交易额占比**63%**
- 北京2026H1招商债权**351.33亿元**，占全国**15.04%**
- 中西部处置周期比东部延长**15%-20%**

## 政策约束

《地方资产管理公司监督管理暂行办法》核心红线：
- 地方AMC原则上不得跨区域经营
- 近三年年均收购金融不良资产投资额占新增投资额比重不低于**30%**
- 融入资金余额不得超过净资产的**三倍**
- 对全部关联方债权余额不得超过上季末净资产的**50%**
