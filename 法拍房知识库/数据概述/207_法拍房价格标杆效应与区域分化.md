# 法拍房价格标杆效应：对楼市四大领域的影响

| 指标 | 数据 |
|------|------|

### 北京市场

- 住宅清仓率54.9%
- 通州、东城去化领先

## 核心机制：价格标杆效应
法拍房最大影响**不是交易体量**，而是形成**片区价格标杆**：
- 6-7折成交 → 成为新参考价
- 购房者压低报价 → 二手房业主被迫降价
- 形成**负向循环**：预期走低→交易停滞→断供→新法拍房入市

## 四大领域影响
1. **二手房** — 区域估值被拉低，"有价无市"
2. **新房** — 三四线法拍房占新房成交20%+
3. **金融体系** — 抵押物缩水→不良贷款风险攀升→银行收紧信贷
4. **个人** — 法拍后剩余债务仍需承担，征信受损

## 国际经验
- **日本**：放任出清→楼市多年走弱→经济长期低迷
- **美国**：主动干预→4年止跌回升
- **启示**：主动干预、标本兼治才能打破下行循环

## 政策转向
- 从"保交楼"→"保主体"
- **控增量**：弹性还款、停息延贷
- **盘活存量**：国企收储法拍房→保障性住房
