# 四大AMC金规17号文解读与重构路径

## 📜 核心政策

**金规〔2024〕17号**：《金融资产管理公司不良资产业务管理办法》
- 共8章70条
- 不良资产收购 → 管理 → 处置 → 其他业务 → 风险管理 → 监督管理

## 🔑 三大核心变化

1. **拓宽收购范围**：允许收购重组资产+其他已发生信用减值的资产
2. **明确非金融不良标准**：鼓励AMC发挥逆周期调节功能
3. **丰富处置工具箱**：追加投资+轻资产业务+受托处置

## 📊 四大AMC实践数据

| AMC | 关键数据/举措 |
|------|------|
| 中国信达 | "一行一策"中小金融机构改革化险方案 |
| 中国东方 | 5年化解金融风险超7000亿元，收购不良近4000亿元 |
| 中国中信金融资产 | 综合运用多种手段助力风险出清 |
| 中国长城资产 | 收购处置+追加投资+轻资产业务三线并进 |

## 📋 五类可开展轻资产业务

1. 咨询顾问
2. 受托处置不良资产
3. 托管高风险机构
4. 担任破产管理人/清算组成员
5. 担任公司信用类债券受托管理人

## 🔄 三大转型方向

1. 坚守主业，践行职责使命
2. 创新驱动，加快业务转型
3. 强化管理，激发发展活力
