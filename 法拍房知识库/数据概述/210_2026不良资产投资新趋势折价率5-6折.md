# 2026不良资产投资新趋势：核心资产折价率升至5-6折

## 市场动态
- 2026年至今银行挂牌转让不良贷款本息达**1160多亿元**
- 2025年银登中心挂牌突破3500亿元（+78%）
- 超20家消金公司合计挂牌逼近400亿元
- 远东集团牵头成立特殊资产中心，四大AMC齐聚

## 法拍市场Q1数据
| 指标 | 数据 |
|------|------|
| 挂牌量 | 29.33万套（+7.20%） |
| 成交量 | 4.87万套（+21.75%） |
| 成交率 | 16.61% |
| 平均变现折扣 | 73.66%（历史低位） |
| 流拍率 | 83%+ |

## 四大投资机会
1. **住宅法拍** — 长三角/珠三角核心城市，清仓率超60%，折价率35-40%
2. **商业地产重整** — 烂尾楼盘活，周期3-5年
3. **上市公司破产重整** — 事件驱动，翻倍级收益
4. **不良ABS** — 标准化产品，年化6-10%

## 四大风险
1. 法律风险（债权瑕疵、权属争议）
2. 流动性风险（资金占用2-5年）
3. 估值风险（回收率不及预期）
4. 政策风险（监管收紧）

## 周期判断
> 当前处于不良资产周期"中后段"——供给持续放量，折价率接近历史低位，对长期资金是布局窗口期。
