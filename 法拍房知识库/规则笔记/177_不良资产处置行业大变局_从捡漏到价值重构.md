# 行业变局 — 不良资产处置从"捡漏"到"专业价值重构"

> 来源文章：[[规则笔记/177_不良资产处置行业大变局_从捡漏到价值重构]]
> 数据截止：2025 年年报 + 2026 年 6 月

## 核心结论

**行业已从"捡漏套利"模式，全面进入"专业价值重构"阶段。**

- 过去：银行打折甩卖 → 买家低价买入 → 简单追偿 → 赚差价
- 现在：精细化尽调 → 投行化重组 → 产业赋能 → 价值重塑

## 标志性事件：中原银行 248 亿特殊资产处置

| 项目 | 详情 |
|------|------|
| 总规模 | **248.65 亿元**（72 个分项债权） |
| 平台 | 北京产权交易所 |
| 披露期 | 2026.6.22—2027.6.18 |
| 处置方 | 中国东方资产深圳分公司（AMC） |

### 资产类型分布
- **地产链条相关**（住宅+商业+开发）占绝对主力
- 涉及蓝光、融创、景瑞、建业等知名房企
- 个人房抵贷集中于京、沪、蓉、宁

## 行业三大趋势

| 维度 | 趋势 | 标志事件 |
|------|------|---------|
| **政策端** | 立法升级（部门规章 → 行政法规） | 《金融资产管理公司管理办法》纳入 2026 年立法计划 |
| **投资端** | 处置 → "处置 + 投资 + 盘活" | 三大 AMC 联手上海科创百亿投资 |
| **供给端** | 房企及金融机构不良资产供给持续增加 | 行业"弹药"充足 |

## 银行不良率结构特征

### 中原银行 2025 数据（典型样本）
- 总量：不良率 1.96%（同比 -0.06pct，**结构恶化但总量改善**）
- 行业 TOP 3：
  - 房地产 5.89%
  - 农林牧渔 5.73%
  - 住宿和餐饮 5.41%
- 资产减值损失 4 年连涨：111.9 → 125.6 → 128.86 → 137.15 亿

## 投行化处置模式

### 传统模式（已过时）
- 折扣买入 → 简单追偿 → 一卖了之
- 处置周期 1-3 年
- 收益来源：差价

### 投行化模式（新趋势）
- 深度尽调 → 债务重组 → 债转股 → 破产重整
- 处置周期 5-10 年全周期
- 收益来源：**卖重组价值 + 产业赋能 + 资产证券化**

## 监管时间线

- **2024.11**：发布《金融资产管理公司不良资产业务管理办法》（8 章 70 条）
- **2026.5.31**：国家金融监管总局明确 AMC 可直接对债权类资产催收
- **2026 年**：陆家嘴论坛释放金融支持房地产最强信号
- **2026 年**：《金融资产管理公司管理办法》纳入立法计划

## 对法拍市场的传导

1. **供给增加**：银行主动出清 → 推升 AMC 包总量 → 法拍端房源增多
2. **价值重估**：起拍价不再单一追求 7 折定价，而需考虑重组价值
3. **专业门槛**：普通投资者难以尽调复杂资产 → 机构化趋势加速
4. **资金需求**：大额包需要产业资本 + 银行系资金联动
