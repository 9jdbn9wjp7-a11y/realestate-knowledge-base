# 资金门槛

## 资金要求
- 多数法拍房**不支持按揭贷款**
- 竞拍成功后 **7 日内一次性付清全款**
- 保证金已押入，尾款凑不上 → 保证金不退

## 解决方案
- 竞拍前确认是否支持贷款 → [[规则笔记/法拍房贷款规则]]
- 提前准备垫资渠道 → [[规则笔记/法拍房融资垫资]]
- 预留足够现金，不要满杠杆操作
