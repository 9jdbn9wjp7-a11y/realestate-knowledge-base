# 流拍不能撤销拍卖——(2023)最高法执复20号

## 案情概要
青海高院以评估价5539.66万元一拍，流拍后降5%至5262.68万元二拍，再次流拍。申请执行人认为降幅不合理，要求撤销拍卖重新进行。

## 申请执行人的三大理由
1. 两次拍卖降价幅度不一致（"同案不同判"）
2. 要求二拍降20%被拒绝，只降了5%
3. 变相强迫以物抵债：要么接盘近2000万超债权对价，要么终结执行

## 最高法院核心裁判

### 流拍 ≠ 可以撤销
- 撤销拍卖的前提是**拍卖已经成交**
- 司法拍卖公告是**要约邀请**（民法典第473条）
- 流拍 = 无人发出要约 → 公告自动失效
- 标的物权属不发生改变，无需专门撤销

### 网拍规定第31条的6种撤拍情形
全部以"拍卖成交"为前提，流拍状态下不适用。

## 实务要点
1. **流拍不能撤销**，只能走后续程序
2. 降价幅度（5%或20%）属法院合议庭裁量范围
3. 流拍后救济路径：接受以物抵债 或 申请其他执行措施
4. 关注法院的**降价合议程序**是否规范（未经合议可提异议）
