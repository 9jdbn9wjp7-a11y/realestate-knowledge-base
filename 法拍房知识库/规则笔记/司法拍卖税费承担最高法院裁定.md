# 司法拍卖税费承担最高法院裁定

## ⚖️ 裁判要旨

当事人可以就司法拍卖税费承担标准进行约定，但无约定或产生争议时，**人民法院应参照民事交易中自主买卖的相关规定确定税费承担标准**。

## 📜 关键法条：《网拍规定》第30条

> 因网络司法拍卖本身形成的税费，应当依照相关法律、行政法规的规定，由相应主体承担；没有规定或者规定不明的，人民法院可以根据法律原则和案件实际情况确定税费承担的相关主体、数额。

## 📊 税费承担主体一览表

| 税种 | 纳税主体 |
|------|------|
| 增值税 | 房屋出卖方 |
| 城建税 | 房屋出卖方 |
| 教育费附加 | 房屋出卖方 |
| 印花税 | 出卖方+买受方 |
| 土地增值税 | 房屋出卖方 |
| 个人所得税 | 房屋出卖方 |
| 奐税 | 房屋买受方 |

## 🚫 关键结论：竞买公告"税费全由买方承担"条款无效

最高法院认为无效，原因：
1. **税收法定原则**：纳税义务人身份不能因拍卖公告条款转移
2. **国税函〔2005〕869号**：法院应协助税务机关优先从拍卖收入中征收税款

## ⚠️ 竞买人4大注意要点

1. 买受人更容易成为缴税义务的"买单人"（税务机关+房管部门制约不对称）
2. 被执行人查无下落时，欠缴税费极可能由买受人买单
3. 无人缴纳税费则产权过户无法完成
4. 可主张适用网拍规定第30条，对抗"税费全由买方承担"条款

## 📌 三个延伸裁判规则

1. 买卖合同对税费承担有明确约定 → 法院认可效力
2. 买卖双方约定税费由一方实际承担 → 系对民事权利义务的处分，法律不禁止
3. 不动产交易税费金额不确定 → 不能从资产评估中扣除

> 案件来源：最高人民法院（2017）最高法执监324号 — 施维、许惠成执行审查类执行裁定书
