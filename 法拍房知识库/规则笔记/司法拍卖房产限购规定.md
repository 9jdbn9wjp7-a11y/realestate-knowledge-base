# 司法拍卖房产限购规定

> 来源：[[法律实务参考]]｜2022-02-16｜[[《最高人民法院关于人民法院司法拍卖房产竞买人资格若干问题的规定》：买受人不具有购房资格，竞得限购房的，能否按悔拍处理？]]
> 类型：最高法文件 / 法拍新政

## 核心规定

《最高人民法院关于人民法院司法拍卖房产竞买人资格若干问题的规定》自**2022年1月1日**起施行。

**要点：** 司法拍卖房产纳入各地房产限购政策范围，受限购约束的竞买人将不被允许参与司法拍卖房产竞拍，包括**司法拍卖、司法变卖和以物抵债**。

## 三大解决问题

1. **统一规则**：明确将司法拍卖房产纳入限购范围
2. **明确职责**：由法院负责审核购房资质
3. **明确责任**：确定不具备购房资格的后果

## 关键条款

| 条款 | 内容 |
|------|------|
| 第一条 | 受限购政策约束的竞买人申请参与竞拍，法院不予准许 |
| 第三条 | 成交后、出具成交裁定书前，法院审核买受人购房资格证明 |
| 第四条 | 虚构购房资格成交的，可被主张拍卖无效并承担赔偿责任 |
| 第五条、第八条 | 以物抵债、司法变卖也纳入限购 |

## 实务适用

### 审核程序
- 竞买人承诺具备购房资格即可参加竞买
- 成交后法院对买受人资质进行**形式审查**
- 买受人应对提交材料真实性负责

### 过渡期认定
- 以**竞价成交时间**为准判断限购适用

### 买受人责任
- 不符合购房资格的，法院**不予出具成交裁定书**
- 不能取得房产所有权
- 保证金不予退还
- 情节严重的可罚款、拘留，构成犯罪的追究刑事责任

### 权利冲突
- 债权人不具备购房资格，流拍后**无法实现以物抵债**
- 共有人、承租人等优先购买权与限购政策关系待厘清
