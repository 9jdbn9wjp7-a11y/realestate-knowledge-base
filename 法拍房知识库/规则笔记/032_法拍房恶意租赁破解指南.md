# 032 如何破解法拍房恶意租赁

## 识别恶意租赁信号

- 20年长租 + 一次性付大额租金
- 屋里住着老人（"钉子户策略"）
- 公告未披露租约

## 三步破解

### 第一步：取证核验（3招）

1. **查时间线** → 对比租赁合同、付款凭证、查封裁定
   - 20年长租+一次性付款 → 基本无效套路
   - 不适用《民法典》725条"买卖不破租赁"

2. **查抵押与租赁顺序** → 先抵押后租赁 → 法院可依法去除
   - 即使租约签在查封前，抵押时间更早就排雷

3. **向法院调档** → 确认公告未披露租约
   - 裁判文书网查原房主与租客是否有诉讼
   - 发现"以租抵债"套路

### 第二步：分情形处理

**恶意虚假租约**：
1. 书面催告腾房（留送达凭证）
2. 申请法院协助强制执行
3. 诉讼维权索赔占用费

**合法有效租约**（查封前签+实际占有）：
- 接管租金（裁定书+新房产证通知改缴租）
- 履行合同到期
- 协商补偿金提前解约（书面+公证）

### 第三步：成本防控

1. 预留清退律师费+占用费+空租期成本
2. 全程录音录像+书面通知+缴费单据
3. 竞拍前查物业/不动产中心租约信息

## 关键法律依据

- 《民法典》725条：买卖不破租赁原则
- **先抵押后租赁** → 租赁对受让人无约束力
- **查封后出租** → 租赁合同无效
