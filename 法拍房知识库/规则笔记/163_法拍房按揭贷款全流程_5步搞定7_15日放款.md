# 法拍房按揭贷款全流程：5步搞定"7-15日"放款

> **原文标题**：《想买法拍房却没全款?按揭贷款完全可以办!》
> **原文来源**：房融堂
> **转载来源**：立德法拍（小法乐拍 m.xflapp.com）

## 核心问题

必须在法院规定时间内交齐全款，否则法院会没收保证金。

**解决方案：法拍房按揭贷款！**

对比抵押贷款优势：
- ✅ 成本低
- ✅ 利率低
- ✅ 贷款年限长

## 法拍贷 vs 二手房贷款

| | 二手房贷款 | 法拍房贷款 |
|---|---|---|
| 流程 | 房屋抵押在前，银行放款在后 | 银行放款在前，房屋抵押在后 |
| 风险 | 低 | 银行承担风险，需第三方担保 |

## 5步法拍房按揭流程

### 01 贷款预审（最重要）
- 银行需在 **法院规定的7-15日** 内将尾款打入法院指定账户
- 逾期不付 → 竞拍前的保证金被没收
- **提前预审非常重要！**

### 02 准备资料
1. 竞拍房源的房产公告、竞拍详情
2. 拍卖保证金付款凭证
3. 竞拍成功确认书
4. 身份证、户口本、结婚证（离婚证/离婚协议）
5. 单位收入证明
6. 个人征信报告、个人银行近一年流水

### 03 出具保函
- 银行贷款审批通过后，要想银行在 **7-15个工作日内发放款**
- 竞买人需提前委托 **担保公司做阶段性担保**
- 担保公司出具《担保函》给银行 → 银行放款到法院指定账户

### 04 贷款额度
- 银行按揭最高额度 **7成**
- 不一定都能贷到这么高，根据资料、财产证明、还款能力判定

### 05 贷款流程（完整链）
```
贷款预审 → 审核通过、参与竞拍 → 拍卖成交 → 
第三方阶段性担保 → 银行发放贷款、支付法院 → 
办理房产过户及抵押
```

## 关键时间节点

| 节点 | 时限 |
|------|------|
| 银行放款至法院 | 7-15日（法院规定） |
| 担保公司出保函 | 7-15个工作日 |
| 提前预审建议 | 至少开拍前 3-5 天 |

## 实操建议

**委托专业的法拍机构助拍**——清晰知道自己能贷多少款、需准备多少资金，竞价时心中有数。
