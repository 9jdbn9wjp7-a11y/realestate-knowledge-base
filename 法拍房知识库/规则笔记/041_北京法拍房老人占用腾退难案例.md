# 北京法拍房腾退案例：八旬老人强占不搬

## 案件概要
- **法院**：北京平谷法院
- **法官**：原婷婷（巾帼执行团队）
- **被执行人**：王某，欠债近千万元
- **占用人**：王某的八旬父母

## 腾退过程（五次交锋）

### 1. 电话联系 → "我们死也不搬"
老人情绪激动，坚称"房子是我们出钱买的"

### 2. 登门面谈 → 申请供暖季后搬
老太太："冬天太冷了，过了供暖季再搬"
→ 法官彰显善意文明执行，准予请求

### 3. 主动到局 → 再求延期
老太太含泪求助："子女都不管我们，有处平房在诉讼中，判了就走"
→ 经查属实，再给宽限

### 4. 多方协调
法官联合民事法官了解进度，联合买房人帮老人找租处

### 5. 果断执行 → 下达罚款决定
> "善意有底线、宽容有限度"

老人自知理亏，在交纳罚款截止日前主动腾退。

## 核心启示
1. **"年迈"不是"老赖"的护身符**
2. 善意文明执行≠无限期容忍
3. 罚款是一种有效的"惩戒预警"
4. 穷尽善意后必须果断启动强制程序
