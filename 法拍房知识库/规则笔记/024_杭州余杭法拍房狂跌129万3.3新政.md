# 24 - 杭州"3.3新政"：首套法拍房狂跌129万+16套紧急撤回

> 原文发布：2021-03 | 抓取日期：2026-07-15  
> 核心事件：**新政后首套法拍房低于评估价 129 万**，**16 套紧急撤回**（含 1 套紧急还债撤回）

## 新政背景

**2021-03-03** 杭州市住保房管局发布《关于进一步规范房地产市场秩序的通知》：
1. 加强住房限购
2. 严防经营贷、消费贷违规流入房地产
3. 打击中介违法违规
4. 房地产市场信息发布

**核心**：参与本市限购范围内住房司法拍卖的竞买人，**须符合本市住房限购政策**。

## 标志性案例：里东山弄首拍

| 项目 | 数据 |
|------|:----:|
| 小区 | 杭州西湖区里东山弄 |
| 面积 | 78.45 m² |
| 楼层 | 5 层中的 3 楼 |
| 位置 | 浙大玉泉校区附近、西湖周边 |
| 起拍价 | 455 万元 |
| 评估价 | 650 万元 |
| 围观次数 | 2 万+ |
| 报名人数 | 7 人 |
| 竞价轮次 | 32 轮 |
| **成交价** | **521 万元** |
| **低于评估价** | **129 万元（约 7 折）** |
| 折合单价 | 6.64 万元/m² |

→ 与二手房市场成交价（6.5-7.5 万/m²）**基本相当**

## 16 套紧急撤回详情

### 撤回原因

| 原因 | 占比 |
|------|:----:|
| 政策调整 / 限购调整 | 多数 |
| **被执行人全部履行债务** | 1 套（房主紧急还债） |
| 拆迁规划红线 | 1 套 |

### 典型撤回案例

**1. 余杭崇贤陆家桥新苑**
- 3 月 3 日 10:00 开拍
- **已进行 4 轮竞价** → 临时撤回

**2. 江干区丁桥颐景园 4 幢 1 单元 701 室**
- 86.54 m² / 起拍 181 万
- **围观 25157 次 / 12 人报名**
- 3 月 4 日 10:00 拍卖中止
- 中止原因：房产有新的限购政策需明确

**3. 临平金都夏宫西荷院 5 幢 2 单元 1901 室**
- 撤回原因：**被执行人全部履行了法律文书确定的金钱债务**
- **房主紧急还清债务**（担心成交价远不如预期）

**4. 江干区罗兰春天**
- 3 月 3 日 10:00 开拍
- 3 人报名 → 临时撤回

## 新政前后对比

| 维度 | 新政前 | 新政后 |
|------|--------|--------|
| 成交价 vs 评估价 | **平均溢价 30%** | **大幅低于评估价** |
| 卖家态度 | 高价挂牌 | **担心成交价低，紧急还债撤回** |
| 房源数量 | 供应充足 | **16 套紧急撤回** |
| 灰色产业 | "抬价"灰产 | **大幅压缩** |

## 典型高溢价标的（新政前）

- 起拍价 1322 万的寰宇天下
- 起拍价 1606 万的青山湖玫瑰园
- 起拍价 2000 万的万丽璞丽
- 起拍价 2250 万的水晶澜轩

## 灰色产业链曝光

### 抬价灰产
- 房子拍卖成交价越高 → 债务人还债负担减少
- 团队**利用买来的身份证信息参与竞拍**
- 专为抬高竞拍价 → 再向债务人索要"中介费"

### 强占灰产
- 安排"债权人"或租客**强占房屋**
- 逼迫买受人缴纳"腾退费"

**新政效果**：
- ✅ 限购要求大幅提高竞拍门槛
- ✅ 外地身份证操纵竞拍几乎不可能
- ✅ 法院承担腾退责任

## 专家观点

> "将法拍房纳入限购政策后，这种**成交价格低于评估价**的现象或将会继续出现。"

## 对南京市场的启示

1. 杭州 3.3 新政是"价值回归"标志性事件
2. 限购令让"虚高法拍房"原形毕露
3. 部分卖家担心价格下行 → **主动撤回** / **紧急还债**
4. 南京 2 万套法拍房规模 → 若跟进限购，可能出现类似震荡
5. **新政前**是卖方市场（高溢价）→ **新政后**快速回归买方市场
6. 真正的"捡漏机会"出现在**新政落地后 1-2 个月**
