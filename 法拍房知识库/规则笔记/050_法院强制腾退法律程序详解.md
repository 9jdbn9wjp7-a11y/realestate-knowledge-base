# 一文读懂法院如何依法强制腾退

> 来源：[[印山资产]] | 日期：2024-12-17 | 北京丰台区人民法院转载

## 什么案件可以申请强制腾退？

### 类型一：司法拍卖后的腾退
被执行人不履行金钱给付义务，法院对其不动产进行司法拍卖后，可强制腾退。

### 类型二：判决直接确定腾退义务
租赁合同、房屋买卖合同、继承类纠纷等，生效法律文书确定腾退义务而拒不履行的。

---

## 强制腾退四步流程

### 第一步：依法张贴腾退公告
- 依据 **《民事诉讼法》第二百六十一条**
- 由**院长签发公告**，责令被执行人在指定期间履行
- 载明腾退原因、期限及法律后果

### 第二步：开展充分的现场调查
- 腾退环境、物品存储量
- 有无管制刀具
- 居住人口数量/性别/年龄 → 制定详细腾退预案

### 第三步：制定详尽的腾退预案
梳理案件情况 → 风险评估 → 执行方案 → 突发事件处置流程

### 第四步：实施强制腾退

| 步骤 | 内容 |
|------|------|
| 准备力量 | 车辆、人员调配 |
| 通知到场 | 被执行人或成年家属、基层组织派人参加 |
| 清点物品 | 执法记录仪记录，制作物品清单 |
| 物品保管 | 搬出财物运至指定处所交付 |
| 确认笔录 | 记入笔录，在场人签名或盖章 |

---

## 突发情况处理

### 阻扰执行
- 可处**罚款、拘留**
- 构成犯罪 → **拒不执行判决裁定罪**（《刑法》第313条）
  - 情节严重：3年以下
  - 情节特别严重：3-7年

### 被腾退人突发疾病
法院提前安排医护人员待命，确保及时救治。

---

## 拒不腾退的法律责任

1. **房屋占有使用费**：持续计算至实际腾退日
2. **迟延履行金**：无论是否造成损失都应支付
3. **其他费用**：搬家、医疗保障等均由被执行人承担

## 关键法条速查

| 法条 | 内容 |
|------|------|
| 《民事诉讼法》第261条 | 强制迁出房屋的法律依据与程序 |
| 《民事诉讼法》第114条 | 阻扰执行的法律后果（罚款/拘留） |
| 《刑法》第313条 | 拒不执行判决、裁定罪 |
