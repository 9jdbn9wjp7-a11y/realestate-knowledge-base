# 040 法拍房有占用拒不搬离——申请强制执行详解

## 法律依据

- 裁定送达后**15日内**完成财产移交
- 被执行人/占有人拒不配合 → **强制执行**
- 所有权自裁定送达买受人时转移

## 法院《四书》

1. 拍卖成交确认书
2. 执行裁定书
3. 协助执行书
4. 税务协助执行书

→ 办理过户需四书，但解决占用问题四书作用有限

## 申请强制执行流程

### 申请时效：**二年**
- 从履行期最后一日起算
- 超过二年 → 丧失申请执行权
- 可中止/中断（适用诉讼时效规则）

### 管辖法院
- 第一审人民法院
- 或与第一审法院同级的财产所在地法院

### 执行四步

1. **立案审查** → 审查申请合法性
2. **通知履行** → 先行通知义务人限期自动履行
3. **准备强制执行** → 确定执行人员和时间
4. **实行强制执行** → 表明身份+出示执法证件+执行根据

### 执行时限

| 类型 | 时限 |
|------|------|
| 有财产可供执行 | 6个月内执结 |
| 非诉执行案件 | 3个月内执结 |
| 特殊需延长 | 报院长批准，期满前5日内申请 |

## 其他惩戒方式

- **拘留/罚款**
- **失信被执行人名单**
- **拒不执行判决裁定罪**（情节严重）

## 对执行有异议

→ 填写执行异议书递交法院

## 核心要点速查

| 要点 | 内容 |
|------|------|
| 交付时限 | 裁定送达后15日内 |
| 申请执行时效 | 2年 |
| 执行管辖 | 第一审或财产所在地同级法院 |
| 有财产案件 | 6个月执结 |
| 非诉案件 | 3个月执结 |
| 惩戒 | 拘留/罚款/失信名单/拒不执行罪 |
