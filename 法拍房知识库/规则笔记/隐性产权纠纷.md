# 隐性产权纠纷

## 常见问题
- 原业主债务复杂，存在隐形共有人
- 抵押未解押
- 多重查封
- 小产权/违建

## 后果
- 过户困难 → 交钱了拿不到房产证
- 办证困难 → 资产等于被冻结
- 需要打官司才能确认产权 → 时间、金钱成本

## 防范
- 查册：到不动产登记中心查产权状态
- 查裁定书：看法院判决中是否提到共有产权
- 调查原业主婚姻状态（婚内财产涉及配偶权利）
