# 法拍房按揭7步流程详解：从竞拍到抵押登记全攻略

> **原文标题**：《法拍房"真香"秘籍 | 贷款篇》
> **原文来源**：思源拍卖
> **转载来源**：小法乐拍（m.xflapp.com，鼓角科技转载）

## 核心结论

法拍房和二手房、新房一样，大部分可以按揭贷款。

⚠️ 贷款类型均为 **商业贷款**，暂不支持公积金贷款。

⚠️ 竞买人要在竞拍前对财务、信用等做准确评估，慎重考虑自身承受能力。

## 当前主流贷款方式

**司法拍卖平台一键贷款服务**（阿里拍卖、京东拍卖提供）：
- 法院给标的物打标"可贷款"或"一键贷款"
- 平台资产交易服务中包含贷款服务
- 建议 **至少在开拍前 3-5 天联系银行**
- 部分标的物支持线上贷款申请

## 7步法拍按揭完整流程

### 第一步：提交资料
**取得法院拍卖房竞拍号后**，在公告后一周内与银行联系申请贷款，逾期不予办理。

**所需材料**：
- 借款人本人及配偶有效身份证件
- 婚姻证明（结婚证、离婚证或离婚协议）
- 详细版个人征信报告
- 收入证明材料（收入证明、银行流水、经营流水等）

⚠️ **可办理贷款的房产类型**：仅限 **两证齐全、权属清晰** 的住房、商铺

### 第二步：贷款资格初审
- 银行客户经理接受贷款申请并收集全部资料
- 银行初审通过后通知 → 到网点领取《贷款意向书》
- 通过材料确认贷款成数、贷款资格

**重要提醒**：
- 🔸 已婚人士需夫妻双方资料，面签需双方带原件现场签字
- 🔸 贷款额度最高可达 **七成**（各地有差异）
- 🔸 银行流水最好准备 **1年收入流水**

### 第三步：竞拍成功
- 报名参拍，预缴保证金
- 网拍系统冻结竞买人淘宝账户内资金作为应缴保证金
- 拍卖结束后未能竞得者冻结的保证金自动解冻
- 竞拍成功后，原冻结保证金自动转入法院指定账户
- 拍卖的尾款在 **7-15个工作日** 内转入法院指定账户

### 第四步：银行面签复审
竞拍成功后到银行面签复审，携带：
1. 借款人本人及配偶（如有）有效身份证件、婚姻状况证明、征信查询授权书、收入证明材料
2. 拍卖公告
3. 拍卖物评估报告
4. 由保证人提交拍卖物现场调查情况说明（出租、欠费等）
5. 银行贷款审批过程中其它需要的资料

### 第五步：银行放款审批
- 拍卖成交后，将首付款缴纳至法院指定收款账户
- 将转账截图发至银行客户经理
- 买受人未取得法律文书，银行在落实抵押权前发放按揭贷款
- 需凭 **担保公司出具的担保函** 担保放款
- 银行贷款审批通过后 → 前往合作担保公司 → 签署担保协议 + 支付担保费用 → 担保公司出具担保函 → 银行放款

### 第六步：协调解封、解押、过户
- 解封房产，在法院办理解封手续
- 银行尾款支付到法院账户，保存好凭证原件
- 与法院执行局相关人员沟通并办理手续
- **确保拿到"四书"**：
  1. 拍卖成交确认书
  2. 民事裁定书
  3. 协助执行书
  4. 税务协执书
- 去房屋所在房管局解押房产
- 缴纳各种税费后完成过户手续
- 正式拿到房产证后收房

### 第七步：办理房屋抵押登记
- 领取新房产证，办理房屋抵押登记
- 银行领取他项权证完成归档手续
- 购房人凭身份证原件到银行领取借款合同及产权证复印件
- 后面开始还款

## 关键时间节点

| 节点 | 时限 |
|------|------|
| 银行联系申请贷款 | 取得竞拍号后 **一周内** |
| 联系银行建议时间 | 开拍前 **3-5 天** |
| 银行流水准备 | **1年** |
| 尾款转入法院 | 7-15个工作日 |
| 担保公司出保函 | 审批后即时 |
