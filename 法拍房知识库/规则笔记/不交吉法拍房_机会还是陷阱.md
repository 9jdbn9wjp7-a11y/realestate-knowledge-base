# 不交吉法拍房_机会还是陷阱

> 来源：南方周末（2020-12-03）
> [[038_不交吉法拍房机会还是陷阱|原文存档]]

## 核心概念

### "不交吉"
- 广东地区用语
- 指 **未腾空就被法院拍卖的房屋**
- 法院拍卖后 **不负责腾空交付**
- 买受人需通过另案诉讼或其他途径解决清场

### "交吉"
- 原居住人在过户前腾空搬离
- 不搬离的由法院负责腾空交付
- 买受人能及时收房入住

## 法律要点

### "先清空再拍卖"缺乏依据
- 最高法院规定：**确权裁定书送达后15天内清空房屋**
- 但 **未规定拍卖前必须腾空**
- 这就给"不交吉"拍卖留下了空间

### 警方处理方式
- 通常认定为"房产纠纷"
- 建议协商解决或诉讼解决
- 不作为刑事案件处理
- 买受人难以通过报警解决入住问题

## 核心案例：杨斌砸门事件

- 当事人：前检察官/律师杨斌
- 房产：广州越秀区金德大厦19层
- 成交价：近94万元
- 问题：租客拒不搬走，破锁换锁循环
- 极端行为：2020年9月11日用锤子砸门
- 结果：警方仍认为是房产纠纷

## 风险清单

| 风险 | 说明 |
|------|------|
| 合法租户 | 买卖不破租赁，租期内无法入住 |
| 原业主占用 | 拒不搬离，需申请强制执行 |
| 其他债权人 | 可能被其他债权人占有 |
| 司法救济困难 | 警方不管，诉讼周期长 |
| 私力救济风险 | 买受人可能被迫采取极端措施 |

## 实务建议

1. **务必确认公告**：是否标注"不交吉"/"不负责清场"
2. **普通购房者**：选择"交吉"拍卖的法拍房
3. **职业投资人**：评估清场难度后参拍，预留清场成本
4. **不要迷信低价**：清场时间和成本可能远超价差
