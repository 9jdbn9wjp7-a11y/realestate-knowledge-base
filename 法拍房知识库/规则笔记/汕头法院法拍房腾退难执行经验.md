# 汕头法院法拍房腾退难执行经验

> 来源：汕头市中级人民法院（2022年11月）
> [[037_亮剑执行之难汕头法院聚焦法拍房腾退难|原文存档]]

## 核心经验：三大举措

### 1. 集中强迁专项行动
- 金平法院2022年4-8月对 **15宗** 未交付法拍房集中研判
- 责任落实到各合议庭和承办人
- 每案出具强迁方案 + 风险预判
- 联合公安、街道居委、卫健部门
- **15宗全部顺利交付**

### 2. 善意文明执行
- 思想工作与强制措施相结合
- 案例：前妻+租户占用 → 买受人续租，现场另签合同
- 原则性和灵活性相结合
- 最大限度减少对被执行人家属的损害

### 3. 社会各界监督
- 全程录音录像
- 邀请人大代表、政协委员、检察院见证
- 媒体跟踪报道
- 确保公开、透明、文明

## 典型案例

### 李某德/麦某源两处房产强制退迁
- 金湖路房产 + 牡丹园房产
- 被执行人拒不退迁 → 法院张贴强迁公告
- 执行法官耐心劝解无效 → 启动强制退迁
- 到达现场后发现已自行撤离
- 专业开锁 + 录音录像 + 拍照取证
- 一个下午完成两处房产交付

## 执行联动机制

| 部门 | 职责 |
|------|------|
| 公安 | 现场安全保障 |
| 街道居委 | 协助协调 |
| 卫健部门 | 应对健康突发状况 |
| 人大代表/政协委员 | 见证监督 |
| 检察院 | 法律监督 |
| 媒体 | 公开透明 |
