# 不良资产清收处置62招全流程

> 来源：融信风控 | 转载：搜狐 | 课程：2026年8月大连

## 核心框架

### 第一部分：清收技巧全面提升
1. **自主清收**：攻心谈判、时效中断、贷款重组
2. **诉讼清收**：财产保全16条线索、破解立案难
3. **执行变现**：追加被执行人、拒执罪、一拍二拍策略
4. **破产保护**：执转破、逃废债防范
5. **以物抵债**：协议vs裁定抵债、税费规避

### 第二部分：六十二招（11大类）

| 大类 | 招数 | 核心 |
|------|------|------|
| 执行管辖 | 1-6 | 财产所在地、移送处置权 |
| 执行措施 | 7-12 | 拍卖变卖、网络通查 |
| 主体扩大 | 13-18 | 出资不实、未经清算追责 |
| 执行分配 | 19-24 | 优先权、刑民交叉 |
| 执行异议 | 25-30 | 异议之诉、继续执行保险 |
| 执行救济 | 31-36 | 执转破、国家赔偿 |

| 处置提升 | 55-58 | 债权置换、代位权 |
| 处置特别 | 59-62 | 在建工程——"政府支持是关键" |
