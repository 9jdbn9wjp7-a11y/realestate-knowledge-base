# 不良资产行业术语大全

## 📚 基础术语

| 术语 | 定义 |
|------|------|
| 剥离卷 | 银行提供交接的项目全部档案材料 |
| 阅卷 | 对剥离卷的阅读分析 |
| 网搜 | 针对项目的网上搜索 |
| 现场 | 债务人经营场所及资产所在地 |
| 本案诉 | 前手债权人的诉讼和执行情况 |
| 涉诉 | 所有债务人及关联企业的其他诉讼执行 |
| 干系人 | 利益受影响的个人或组织 |

## 🏗️ 土地分类术语

| 术语 | 定义 |
|------|------|
| 出让地 | 需支付土地出让金的有偿使用 |
| 划拨地 | 无需出让金的公益等用地，转出让需审批 |
| 集体土地 | 农民集体所有土地 |
| 收储 | 国家优先购买+土地整理→储备用地（仅净地可收储） |

## ⚖️ 法律术语

| 术语 | 定义 |
|------|------|
| 共益债 | 破产程序中为共同利益所负担的债务 |
| 首封 | 首轮查封法院 |
| 轮候查封 | 排队等候，解除后自动转化正式查封 |
| 限高令 | 对"老赖"限制高消费 |
| 他项权证 | 抵押权人持有的权利证书 |
| 潜在债务人 | 尽调发现的额外可能承担债务的主体 |
| 隐形债务 | 未在财务资料中体现的或有债务 |

## 💼 不良资产业务术语

| 术语 | 定义 |
|------|------|
| 收购处置业务 | 收购→管理→经营→处置→回收现金 |
| 收购重组业务 | 收购时确定提升偿债能力的技术手段 |
| 尽职调查 | 对拟收购资产状况的调查分析 |
| 资产定价 | 对收购资产经营价值的估算 |
| 资产证券化 | 现金流资产→结构性重组→可流通证券 |
| 可疑类贷款 | 无法足额偿还，即使执行担保也有较大损失 |
| 损失类贷款 | 所有措施后仍无法收回 |
| 不良资产剥离 | 商业银行不良→四家AMC（华融/信达/东方/长城） |
| 贷款核销 | 银行内部冲销不良，"账销案存" |
| 逃废债 | 有履行能力而不尽力履行，强调主观故意 |

## 🧭 知识体系五维框架

1. **入门**：行业历史、形成机制、定价原则、税负知识、法律风险
2. **投资**：投资结构、收购要点、投前尽调、风险控制
3. **处置**：市场现状、处置逻辑、处置创新、成功案例
4. **破产**：重组、财产分配、逃废债、涉税法律、债权保护
5. **尽调**：现场尽调、资产估值、各类资产尽调要点
