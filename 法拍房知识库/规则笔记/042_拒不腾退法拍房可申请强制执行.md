# 【法官说法】拒不腾退法拍房，可申请强制执行！

> 来源：澎湃新闻·政务号 | 法官说法【2024】72

## 核心法律依据

### 《拍卖变卖规定》第二十七条
> 人民法院裁定拍卖成交后，除有依法不能移交的情形外，应当于裁定送达后**十五日内**移交买受人。被执行人或者第三人占有应当移交而拒不移交的，**强制执行**。

### 《刑法》第三百一十三条 拒不执行判决、裁定罪
- 情节严重：三年以下有期徒刑、拘役或罚金
- 情节特别严重：三年以上七年以下有期徒刑

## 关键判决

高某竞得法拍房 → 原房主张某拒不搬出 → 高某另案提起民事诉讼

**法院裁定：驳回起诉！**
- 理由：属于**执行程序中的问题**，不是新的民事侵权
- 正确做法：向原执行法院**申请强制执行**（不是另案起诉）
- 结果：强制执行后，高某顺利入住

## 实操路径

```
竞拍成功 → 取得裁定书 → 办理过户 
→ 遇到占用人拒搬 
→ 向原执行法院申请强制执行（非另案起诉）
→ 法院强制清场交付
```

## 法官核心观点
1. 原房主非法占有拒不移交 = 执行程序问题，**不是新民事侵权**
2. 不宜另案提起排除妨害之诉 → **直接申请强制执行**
3. 法院将逐步升级强制措施：限期搬离 → 罚款/拘留 → 追究刑事责任
