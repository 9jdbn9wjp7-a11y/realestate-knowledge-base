# 栖霞-中骏东原璟阅 — 3折甩卖24套

## 基本信息
- **区域**：栖霞区龙潭街道后曹路6号
- **地址**：中骏东原璟阅小区 3栋、5栋高层
- **面积**：124.93㎡ / 127.07㎡（大三房）
- **起拍价**：70万-73万/套
- **评估价/市场价**：新房销许价约1.8万/㎡ → 备案总价230万+
- **折扣**：约 3 折（起拍单价 5600-5800元/㎡ vs 销许价 18000元/㎡）
- **拍卖平台**：阿里拍卖
- **拍卖日期**：2026年7月

## ⚠️ 特殊风险
- **查封+抵押**：房源存在查封、抵押情况
- **长期空置**：内部无入住痕迹，阳台未封闭，室内有渗水痕迹
- **隐性成本**：可能拖欠物业费、水电费等
- **开发商暴雷背景**：南京骏原房地产（中骏+东原合资）与龙信建设集团的工程款纠纷，44套房源被移送拍卖/变卖
- ⚠️ **捡漏空间存疑**：起拍单价5600元/㎡，但小区二手房实际成交均价仅5271元/㎡，起拍价反而高于真实成交价，几乎不存在捡漏空间

## 💡 亮点
- **批量法拍**：一次上架24套，开发商自持库存新房被查封拍卖
- **价格缩水**：从230万+缩水到70万起拍，单套至少缩水160万
- **双TOP50房企联手**：中骏集团 + 东原地产（迪马股份旗下）合资打造，当年被视为龙潭新城标杆楼盘
- **板块兴衰缩影**：2018年拿地 → 2019年首开 → 2020年竣工 → 2021年登记 → 2026年被查封拍卖

## 📝 脚本素材
- **反转叙事**：表面看3折甩卖是捡漏，实际起拍价比二手房成交价还高 → "你以为的骨折价，其实是市场价"
- **开发商暴雷链**：工程款纠纷 → 44套被查封 → 开发商自持库存变法拍房 → 双TOP50房企的龙潭标杆项目陨落
- **板块困境**：龙潭新城当年被看好，地铁S5号线规划中，但至今未通车，交通不便、配套不足导致房价持续下跌
- **法拍 vs 二手房对比**：法拍房带隐性成本（查封、抵押、欠费），同小区二手房5271元/㎡可能更划算
